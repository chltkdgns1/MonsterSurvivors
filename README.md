# 몬스터 서바이벌
----------------------------------------    
* 간략한 게임 소개
  * 유니티를 기반으로 제작함.
  * 뱀파이어서바이벌을 모티브로 하여 개발 진행
  * 몬스터가 많이 나오기 때문에 모바일 환경에서 할 수 있도록 최적화 진행
 ----------------------------------------    
* 최적화
  * 드로우콜 최적화
    * 아틀라스를 사용하여 드로우콜 줄임.
  * 리지드바디 사용
    * 리지드바디를 사용하지 않고 콜라이더만 사용하는 것이 더 느릴 것이라고 생각함.
    * 실제 콜라이더만 사용하는게 더 느림.
  * 오브젝트풀을 사용하여 오브젝트, 생성 삭제 비용 줄임.
  * 프로파일링을 통해서 병목되는 부분 찾아서 처리.   
----------------------------------------    
* 언어
  * C#

* 개발 도구
  * 유니티
